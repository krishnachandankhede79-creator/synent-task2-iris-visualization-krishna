#  Iris Data Visualization Project

## 📌 Problem Statement
The objective of this project is to visualize patterns in the Iris dataset.

## 📊 Dataset
Iris dataset (CSV format)

## 📈 Visualizations
- Bar chart (species count)
- Histogram (sepal length distribution)
- Scatter plot (sepal length vs petal length)

## ⚙️ Tools Used
- Python
- Pandas
- Matplotlib

## ✅ Result
The visualizations show clear patterns and relationships between features of different species.
